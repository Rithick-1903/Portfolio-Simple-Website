# Simple Two Page Portfolio Website

Welcome to my project this is a simple portfolio website title as  **My portfolio website** created using **HTML and basic CSS**. It demonstrates fundamental web development concepts such as page structure, styling, and navigation between multiple pages.

## Pages
- **Home Page** – Displays an introduction and basic information.
- **Portfolio/About Page** – Shows personal details, skills, or projects.

## Technologies Used
- HTML5
- CSS3
